# 中国市场募资打法手册 / China-market fundraising playbook

> 用途:当创始人在**中国大陆市场**募资,或要求**中文话术**时,先读这份。
> 中国募资不是把美国话术翻译一遍——渠道、中介、礼节、数据源都不同。
> 所有 skill 产出中文内容前都应参考本文 + `outreach-ethics.md`。

## 一、最大的差异:渠道是微信,不是 email

- **微信为王。** 美国靠 cold email,中国投资人主要在**微信**沟通。陌生邮件回复率极低。
- **标准路径**:通过引荐 → 加微信 → 发 BP(PDF)→ 约线上/线下见面。
- **加微信本身需要由头**:最好由引荐人拉群或推名片;直接加好友要附一句清晰的自我介绍(见 cold-email 中文话术里的"微信开场")。
- **朋友圈是软触达**:发产品进展、里程碑,让目标投资人"先眼熟你",再正式接触,转化更高。

## 二、关键角色:FA(融资顾问 / 财务顾问)

- 中早期很多创始人通过 **FA** 对接机构。FA 帮你梳理 BP、匹配机构、安排 meeting、把控节奏。
- **何时考虑 FA**:自己人脉够不到目标机构、没时间跑流程、轮次偏大(A 轮及以后)时更常见。
- **代价**:FA 通常收**融资额的 2%–5% 财务顾问费**(具体谈,以实际签约为准,别拍脑袋报数)。早期小额融资不一定划算。
- **注意**:好 FA 是加速器,差 FA 会稀释你和机构的直接关系。FA 不能替代创始人本人的 pitch。

## 三、关系/人情 > 陌生触达

中国语境下 warm intro 的权重比美国更高,纯陌生触达基本无效。高质量引荐来源:

- **被投企业创始人**(机构最信自己投过的人)
- **同行/上下游创始人**
- **校友**——尤其商学院(长江、中欧、湖畔)、名校 EMBA 圈层,人脉密度高
- **前同事 / 前老板**(大厂出身的人脉是硬通货)
- **行业活动 / 路演**——线下见面在中国仍很重要

## 四、礼节与称呼(影响第一印象)

- **称呼要得体**:对投资人常用「X 总」「X 老师」;不确定时用「X 总」较稳妥。
- **语气更谦逊正式**,但不卑微;开门见山讲清楚是谁、做什么、为什么找他。
- **尊重对方时间**,但中国沟通常先寒暄一句再进正题,比美国略"软"一点。
- **BP 文化**:习惯先发**商业计划书(BP,PDF)**,而非美式"先口头 pitch"。BP 要能独立讲清楚故事。

## 五、机构生态:人民币基金 vs 美元基金

- **美元基金**:看大市场、高增长、可对标美股/港股退出;流程更接近硅谷。
- **人民币基金**:更看现金流/盈利路径、确定性;部分**国资背景基金有"返投"要求**(要求被投落地当地、带动当地就业/税收)和更重的合规流程。
- **务必问清基金属性**:它决定了对方的诉求、决策速度和对赌/回购等条款倾向。
- ⚠️ **回购/对赌条款**在人民币基金里较常见,签前务必让创始人找专业律师看,本工具不提供法律意见。

## 六、中国版"找名单"数据源(免费/freemium)

`investor-targeting` 在中国市场可用这些实时查证(替代 Crunchbase 那套):

- **IT桔子(itjuzi.com)** — 中国创投数据库,查机构出手、轮次、赛道
- **烯牛数据、鲸准** — 投融资事件库
- **企查查 / 天眼查** — 查机构工商背景、关联、基金属性(人民币/国资)
- **36氪、投资界(PEdaily)、铅笔道、IT桔子公众号** — 融资新闻,看谁在出手
- **机构官网 / 公众号** — 投资方向、被投案例、联系方式
- 同样规则:**实时查证 + 标注来源 + 不凭记忆编机构名**。

## 七、把美国流程映射到中国(skill 协同)

| 美国流程 | 中国对应 |
|---|---|
| cold email | 微信开场白 + 发 BP(见 cold-email 中文话术) |
| warm intro(可转发邮件) | 引荐人拉微信群 / 推名片 + 引荐语(见 warm-intro 中文话术) |
| investor-targeting 数据源 | IT桔子/企查查/36氪(见上) |
| pitch deck | BP(商业计划书,PDF) |
| 中介 | FA(融资顾问) |

## 八、给 skill 的输出原则

- **不是翻译,是本地化**:语气、称呼、渠道、礼节都要按中国习惯改,而非直译英文模板。
- **默认渠道是微信**,邮件为辅;话术要给"微信版"和"邮件版"两种。
- **数字仍要真实**:夸大 traction、虚构机构/客户的红线,中美一致。
- **法律/条款不给意见**:回购、对赌、返投等一律提示找专业律师。
