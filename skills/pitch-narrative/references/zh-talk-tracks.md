# 募资故事中文话术(一句话 + BP 故事线)/ Chinese pitch-narrative talk tracks

> 面向**中国大陆市场**。先读 `shared/references/china-market-playbook.md`
> 和 `shared/references/outreach-ethics.md`。
> 中国习惯先发 **BP(商业计划书,PDF)**,所以故事要能"独立讲清楚"。
> 不是翻译英文模板,是用中文把故事打磨锋利。数字必须真实。

## 一、一句话定位(微信开场、引荐语、BP 首页都复用)

公式任选,选最清楚的那个:
- **要做的事**:「我们帮 <客户> <做成某事>,无需 <某痛点>。」
- **类比式**:「<某熟悉品类> 的 <你的差异点> 版。」
- **X for Y**:「<已知的东西> 的 <新领域> 版。」(引荐时最好用,一句就能转述——但类比必须诚实)
- **结果导向**:「我们让 <客户> 从 <差状态> 变到 <好状态>。」

规则:外行也能听懂;不堆术语;≤20 字最佳。

## 二、电梯陈述(约 30 秒 / 3–5 句)

一句话定位 + 最硬的一个证据 + 为什么是现在。微信首条、电话开场都用得上。

## 三、BP/募资故事六拍(每拍能独立讲清)

1. **痛点** — 具体客户的真实痛,越具体越好,别讲宏大叙事。
2. **为什么是现在** — 什么变了(技术/行为/政策/成本曲线)让这事今天才成立。**投资人投的是时机**;中国尤其看政策与市场窗口。
3. **切入点/做了什么** — 锋利的小切口,不是五年大平台。先做什么、为什么是"刀"不是"勺"。
4. **牵引力** — 最硬的真实证据(流水、增长、复购、深度合作客户、意向订单)。**讲增速优先于绝对值。**
5. **为什么是我们** — 创始人-赛道匹配:亲历的痛、独到洞察、过往履历。一句具体真话胜过一堆头衔。
6. **本轮融资** — 融多少、这笔钱买什么(具体里程碑)、能把你带到下一轮需要的什么指标。

## 四、打磨方法

- **先口语讲一遍,再提炼**:让创始人随口讲,你把最强的一版镜像回去,砍掉术语。好句子常埋在第三段。
- **逐拍压力测试**:切入点问"为什么先做这";为什么是现在问"2019 年为什么做不了";为什么是我们问"你知道什么别人不知道的"。答不上来的地方就是漏点,如实指出。
- **可转述测试**(质量标准):投资人听一遍后,能不能准确转述给合伙人?中国走引荐+微信转发,**可转发性**尤其关键。优化到"换个人也能替你讲清楚"。

## 五、按对象微调(主线不变,侧重变)

- **美元基金**:重为什么现在 + 大市场 + 增长。
- **人民币基金**:重确定性 + 现金流/盈利路径;若是国资背景,讲清落地价值(就业/税收/返投)。
- **产业/战略投资人**:重你的切口与他主业的协同 + 真实拉力。
- **天使/个人**:重为什么是你 + 洞察;他们投人、决策快。

侧重可变,**事实不可改**。

## 六、硬规则

- **大白话,不堆术语**——清晰=自信,术语=心虚。
- **真实数字**;不靠夸大或虚构的 traction 撑某一拍——去改"真实事实的讲法",别造证据。
- 没有真实的"为什么现在 / 为什么是我们"就如实点出——现在发现漏洞,好过在投资人面前露馅。
- 这套话术是 `cold-email`/`warm-intro`/`cold-call` 中文版和 BP 的**共同源头**,改了要同步。
