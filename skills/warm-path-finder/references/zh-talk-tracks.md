# 找对人 + 找暖路径 中文话术 / Chinese warm-path talk tracks

> 面向**中国大陆市场**。先读 `shared/references/china-market-playbook.md`
> 和 `shared/references/outreach-ethics.md`。
> 中国和美国最大不同:**领英在华已停服**,关系链在微信/脉脉/校友圈里;
> 但要拿欧美资方,X(对方公开账号)和被投创始人引荐照样适用。

## 一、找对人:该找机构里的谁

中国机构同样按赛道/阶段分人,别只盯最大的老板:

- **合伙人 / 董事总经理(MD)**:拍板的人,需要强引荐,适合后期或他亲自看你赛道时。
- **投资总监 / 资深投资经理 / VP**:**真正在外面找项目的人**,通常**最该先接触**——回复快、会带你往上推。别一上来只想搭最大的 GP。
- **投资经理 / 分析师**:负责初筛,能把你递上去。

**怎么找到这个人(公开渠道)**:
- 机构官网/公众号「团队」页 → 对上你赛道的人。
- 搜「<对标公司> <轮次> <机构> 领投/负责人」→ 谁投的同类项目,谁就懂你的赛道。
- 看他在**朋友圈/公众号/播客/行业大会**讲什么——写过你这问题的人就是你的内部支持者。

## 二、Tier 1:你自己的授权数据(最值钱、最合规)

- **你的 Gmail / 邮箱**(已接入的话):搜机构名/人名,看你是不是早就和能连上的人通过信。
- **微信好友 / 群**:这是中国最重要的关系链。手动翻:这个机构的人、**被他们投过的创始人**、和目标同公司/同校的人。被投创始人引荐**最值钱**。
- **聚焦联系人表**:微信导不出 CSV,改用手机通讯录导出 + 名片全能王/CamCard + 校友名单,填进 `references/contact-sheet-template.csv` 交给 skill 比对(详见 `references/exporting-your-data.md`)。
- **脉脉(Maimai)**:中国职场关系网,可看公司、找人、看共同人脉(类似领英的国内替代)。
- **校友圈**:长江/中欧/湖畔、名校 EMBA——人脉密度高,常有现成的群。

## 三、Tier 2:公开网络

- **被投创始人引荐(最强外部路径)**:机构投过的创始人替你说话,分量最重。先找机构 portfolio 里和你相近的公司。
- **X / Twitter**(针对欧美资方):找到合伙人/投资经理,看公开观点,看**私信开没开**(主页有私信按钮就能发)。你确认状态,我帮你写那条 DM 或「评论 + 私信」开场。
- **行业活动 / 路演**:机构的人在哪场分享——线下打个招呼,远胜冷私信。

## 四、Tier 3:你登录后手动查

- **脉脉/领英(若你海外账号还在)看共同人脉**:目标主页看「你们怎么连上的」。
- **X 私信状态**:打开主页看有没有私信图标。

## 五、排序:信任 × 可达性

给每条路径打分,**两项都高的优先**:

- **信任**(对方多信这个引荐人):★★★ 被该机构投过的创始人 / 他信任的同事 > ★★ 受尊敬的同行、前同事、共同投资人 > ★ 泛泛之交。
- **可达性**(你多容易开口、对方多可能答应):★★★ 关系近、今天就能开口 > ★★ 认识但要谨慎开口 > ★ 几乎不熟。

**理想路径 = 你真认识的、被该机构投过的创始人。** 找到最佳一条,交给 `warm-intro` 中文话术(走微信拉群)写引荐请求。

## 六、红线(中美一致)

- 只用**你本人授权/导出**的数据 + **公开**信息。
- 不爬登录后的关系链、不猜/不买私人联系方式去群发。
- 不编造「共同好友」——数据里没有就别说有。
